# Reference Materials (`/references`)

Thư mục này chứa tất cả các tài liệu tham khảo kỹ thuật, bản thiết kế kiến trúc, đặc tả API (API Specifications), tài liệu nghiệp vụ (domain requirements) liên quan đến hệ thống HRMS.

## 📂 Danh mục tài liệu khuyến nghị lưu trữ
- **API Specs**: Các file Swagger/OpenAPI JSON/YAML, tài liệu Postman collection.
- **Database Diagrams**: Sơ đồ Entity-Relationship (ERD), sơ đồ thực thể liên kết dạng ảnh hoặc tài liệu mô tả.
- **Architecture Diagrams**: Sơ đồ phân tầng hệ thống (Clean Architecture diagram), luồng dữ liệu (Data Flow Diagrams - DFD).
- **Domain Docs**: Quy trình nghiệp vụ nhân sự (ví dụ: Quy trình duyệt đơn xin nghỉ phép, quy trình tính lương).
