# Tài liệu Tham khảo Frontend (`/references`)

Thư mục này chứa các tài liệu thiết kế UI/UX, tài liệu đặc tả giao diện và các tài liệu tham khảo kỹ thuật liên quan đến Frontend của hệ thống HRMS.

## 📂 Các tài liệu khuyến nghị lưu trữ:
- **Wireframes & Mockups**: Đường dẫn Figma, ảnh chụp màn hình thiết kế hoặc bản phác thảo giao diện.
- **UI Kit & Design Tokens**: Hướng dẫn bảng màu (HSL), font chữ, khoảng cách (spacing), và các quy chuẩn thiết kế chung (Design System).
- **API integration contracts**: File mẫu JSON request/response và đặc tả luồng xử lý của Client.
