# Scripts Hỗ trợ Phát triển Frontend (`/scripts`)

Thư mục này chứa các kịch bản chạy thử nghiệm giao diện (mock server, static server), linters, build và các công cụ phục vụ tự động hóa quy trình phát triển giao diện của HRMS.

## 📂 Danh mục các script khuyến nghị:
- **Server chạy thử (Development Static Server)**: Các script khởi động máy chủ phục vụ file tĩnh cục bộ (như `http-server`, `browser-sync`).
- **Code Linters & Formatters**: Cấu hình kiểm tra định dạng và chất lượng code CSS/HTML/JS tự động (như ESLint, Stylelint).
- **Build & Package**: Scripts đóng gói ứng dụng (nếu sử dụng bundler như Vite/Webpack).
